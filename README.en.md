# ASISLAB - DEMO

🇺🇸 English | 🇪🇸 [Español](README.md)

---

ASISLAB is a web platform for managing preventive training within a company. It allows the creation of training materials using AI, worker management, and analysis of results through statistics.

## Demo

https://asislab-demo-frontend.vercel.app

> ⚠️ The backend uses a free service. The first load may take a few seconds.

## Features

- 🤖 Automatic AI-powered presentation generation.
- 📝 Automatic quiz creation from presentation slides.
- 👥 Management of workers, departments, and quizzes.
- 📊 Statistics dashboard with performance metrics.
- 📈 Charts to visualize results and progress.
- 🔐 Authentication system with roles (Administrator and Guest).
- 👤 Guests can access limited features.
